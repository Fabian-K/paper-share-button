##&lt;paper-share-button&gt;

Material design: URL sharing system

`<paper-share-button>` is a light Material Design URL sharing system.

### Compatible w/
+ E-mail
+ Facebook
+ Google Plus
+ Twitter
+ Reddit
+ VK
+ Blogger
+ Tumblr
